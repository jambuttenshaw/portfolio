# Weather Dashboard

An interactive weather dashboard that provides detailed weather information for any location, with beautiful visualizations and forecasts.

## Features

- Current weather conditions
- 7-day weather forecast
- Hourly weather details
- Interactive weather charts
- Location search and geolocation
- Favorite locations
- Responsive design for all devices

## Technologies Used

- **Frontend:** Vanilla JavaScript, HTML5, CSS3
- **API:** OpenWeatherMap API
- **Charts:** Chart.js for weather visualizations
- **Styling:** CSS Grid and Flexbox
- **Storage:** LocalStorage for favorites

## API Integration

The app integrates with the OpenWeatherMap API to fetch real-time weather data including:

- Temperature, humidity, wind speed
- Weather conditions and icons
- Sunrise/sunset times
- UV index and air quality

## User Experience

- Clean, intuitive interface
- Fast loading times
- Offline-capable (cached data)
- Accessible design following WCAG guidelines
