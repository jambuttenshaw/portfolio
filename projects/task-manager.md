# Task Management App

A modern task management application that allows teams to collaborate on projects with real-time updates, drag-and-drop functionality, and comprehensive project tracking.

## Features

- Real-time collaboration
- Drag-and-drop task organization
- Team member assignment
- Project timelines and deadlines
- File attachments and comments
- Progress tracking and reporting
- Mobile-responsive design

## Technologies Used

- **Frontend:** Vue.js, Vuex, Vue Router
- **Backend:** Firebase (Firestore, Authentication, Hosting)
- **Real-time:** Socket.io
- **UI Framework:** Vuetify
- **Charts:** Chart.js for progress visualization

## Key Components

### Real-time Updates
Tasks update instantly across all team members' browsers using WebSocket connections.

### Drag & Drop Interface
Intuitive drag-and-drop functionality for moving tasks between different stages.

### Team Collaboration
Assign tasks to team members, add comments, and track progress collaboratively.
